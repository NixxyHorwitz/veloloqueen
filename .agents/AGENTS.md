# Velostar Project Rules

## Auto-Push After Every Change
Setelah setiap perubahan file di project ini, SELALU lakukan git push ke remote origin secara otomatis tanpa perlu diminta oleh user:
```bash
git add -A
git commit -m "<deskripsi singkat perubahan>"
git push origin main
```
